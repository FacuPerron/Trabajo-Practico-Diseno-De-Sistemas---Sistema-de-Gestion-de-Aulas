# Trabajo-Practico-Diseno-De-Sistemas---Sistema-de-Gestion-de-Aulas
Este proyecto consiste en el desarrollo de un sistema para la gestión y asignación de aulas dentro de una institución académica.  El sistema permite administrar reservas de aulas para el dictado de clases, considerando distintos tipos de aulas, disponibilidad horaria y requerimientos específicos de cada curso.
